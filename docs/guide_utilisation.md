# Guide d'utilisation

Ce document expliquera comment installer, configurer et lancer le chatbot local du data center.

1. Installer les dependances Python.
2. Preparer les donnees dans data/raw/.
3. Nettoyer et convertir le dataset.
4. Construire l'index RAG local.
5. Lancer le chat offline.
6. Evaluer les reponses du modele.
